#######################################

Sudoku Game

#######################################

Note: This game was made on Linux OS, using a bash script and cmake to compile the source code, written in C++.

To compile the game simply run in bash:
./build.sh
Then, in order to generate a new game:
./sudoku [dificulty]
where difficulty:
  1 - easy
  2 - normal
  3 - hard
  4 - insane
